# Alec's Coops!

`Alec's Coops!` is a lightweight companion mod for `Alec's Tamework!`.

## Purpose

This mod owns coop asset overrides so `Alec's Tamework!` can keep coop Java/runtime logic without shipping base-game coop asset overrides directly.

## Requirements

- `Alec's Tamework!` `2.7.x`

## Current Contents

- `Server/Farming/Coops/Coop_Chicken.json` override
